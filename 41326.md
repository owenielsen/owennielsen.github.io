## How to Run the Debugger

### Method 1: VS Code (Recommended)
1. Open file: [errorKG/Main.java](errorKG/Main.java)
2. Press **F5** or click Run → Start Debugging
3. Debugger pauses at breakpoints automatically
4. Inspect variables in Variables panel
5. Use **F10** (Step Over) or **F5** (Continue)

### Method 2: Manual Java Debug Server
```bash
# Terminal 1: Start Java with debugging
java -agentlib:jdwp=transport=dt_socket,server=y,suspend=y,address=5005 \
     -cp bin errorKG.Main

# Terminal 2: Connect VS Code to port 5005
# (Debug Adapter listens on port 5005)
```

### Method 3: Command Line Debugging
Use `jdb` (Java Debugger):
```bash
jdb -sourcepath src -classpath bin errorKG.Main
> stop at errorKG.Main:17
> run
> next
> print i
```
